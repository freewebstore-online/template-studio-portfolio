# A Studio — Portfolio Template

A modern bento-grid portfolio template for creative directors and UI/UX designers. Built with HTML, CSS, and vanilla JavaScript.

## Features
- Responsive bento grid layout
- Hero section with stats
- Design philosophy / methodology section
- Project case study slider
- Service offerings grid
- Client logo banner
- Contact form with validation
- Scroll reveal animations
- Mobile-friendly navigation
- Back to top button

## fws Slots
All content sections use `data-fws-slot` attributes for freewebstore template compatibility.

## Deployment
```bash
npx @freewebstore/cli publish
```
